# CareConnect – Client Interaction & Follow-up Tracker
_______________
CareConnect is a simple client interaction and follow-up tracking system designed to simulate real-world user engagement workflows. It helps manage user queries, track interactions, and schedule follow-ups to improve communication and response handling.

## Features
__________________
- Record client interactions and queries
- Track follow-ups and interaction status
- Store structured user information
- Simple and user-friendly interface

## Tech Stack
___________________
- Python
- Flask
- SQLite

## Use Case
____________________
This project simulates how organizations manage client communication, ensuring proper understanding of user needs and maintaining continuous engagement through structured follow-ups.

## How to Run
_______________________
1. Install dependencies:
pip install flask
2. Run the application:
python app.py
3. Open browser:
http://127.0.0.1:5000/
